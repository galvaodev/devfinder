
  

# **DevFinder**

  

 

  

O objetivo realizar pesquisas de usuário do github.

  
> ## Rodar o projeto
> 
- Instalação das dependencias:
`npm install`

- Rodar o projeto na máquina utilizando
`npm dev`

Utilizar node versão: 16.10.0 >
